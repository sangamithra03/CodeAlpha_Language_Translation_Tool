# 🌍 AI Language Translation Tool

## 📌 Project Overview
The AI Language Translation Tool is a simple web application developed using Python, Gradio, and the Deep Translator library. It enables users to translate text from one language to another through an easy-to-use interface.

This project was developed as **Task 1** for the **CodeAlpha Artificial Intelligence Internship**.

---

## ✨ Features

- Translate text into multiple languages
- Automatic source language detection
- User-friendly interface using Gradio
- Fast and accurate translations
- Supports multiple language pairs

---

## 🛠️ Technologies Used

- Python
- Google Colab
- Gradio
- Deep Translator

---

## 🚀 How to Run the Project

1. Open the notebook in Google Colab.
2. Install the required libraries.
3. Run all the cells in order.
4. Open the generated Gradio URL.
5. Enter the text, select the source and target languages, and click **Submit** to view the translation.
